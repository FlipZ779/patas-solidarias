#  Patas Solidárias

Projeto web desenvolvido como parte de uma **atividade de extensão universitária**, com o objetivo de **arrecadar ração para animais resgatados** e apoiar entidades que atuam na proteção de cães e gatos em situação de rua ou vítimas de maus-tratos.

O site busca **conectar estudantes, comunidade e organizações de proteção animal**, facilitando a divulgação da campanha e incentivando a participação da sociedade.

---

#  Objetivo do Projeto

O **Patas Solidárias** foi criado para ajudar na arrecadação de ração destinada a animais domésticos resgatados e acolhidos por instituições de proteção animal.

Além da arrecadação, o projeto também busca:

* Promover a conscientização sobre abandono e maus-tratos de animais
* Incentivar a doação voluntária
* Fortalecer a colaboração entre universidade, alunos e comunidade
* Aumentar a visibilidade das necessidades das ONGs de proteção animal

---

#  Instituição Parceira

O projeto atua em parceria com a **Associação Polly Cães & Gatos**, organização sem fins lucrativos fundada em **2013**, localizada em **Ivinhema – MS**.

A ONG realiza:

* Resgate de animais em situação de abandono
* Acolhimento e tratamento
* Reabilitação
* Encaminhamento para adoção responsável

Todas as doações arrecadadas através do projeto são destinadas à instituição para ajudar na alimentação e manutenção dos animais resgatados.

---

#  Contexto Social

Segundo estimativas da Organização Mundial da Saúde (OMS), o Brasil possui aproximadamente:

* **30 milhões de animais abandonados**
* **20 milhões de cães**
* **10 milhões de gatos**

Muitos desses animais vivem nas ruas enfrentando:

* fome
* doenças
* maus-tratos
* falta de cuidados básicos

As ONGs de proteção animal desempenham um papel essencial no resgate e acolhimento desses animais, porém frequentemente enfrentam **dificuldades financeiras**, especialmente para garantir a alimentação diária.

Este projeto busca contribuir para minimizar esse problema.

#  Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/patas-solidarias.git
```

Entre na pasta:

```bash
cd patas-solidarias
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

O projeto estará disponível em:

```
http://localhost:3000
```
